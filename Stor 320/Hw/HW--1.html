ggplot(data=diamonds,aes(carat,price))+

## Warning: Using shapes for an ordinal variable is not advised
## `geom_smooth()` using method = 'gam' and formula 'y ~ s(x, bs = "cs")'

Four variables in one chart is somewhat messy. But as I set price and carat as x and y, the linear increasing curve generated, indicating price and carat is positively related. And shape of the point is used to represent how well the cut is and the color represents the clarity of the diamond. And an interesting fact is when we look the first chart, it seems the worse the clarity is, the higher the relative price will be, since the median value for IF is generally less than any other clarity status. However by the second chart we could find the reason, since very few diamonds with highest clarity(yellow color) have high carat amount but less clarity diamonds would have a larger portion of high carat units. Because more carat tends to bid up the price, so the median value for not-IF diamonds would increase by having more numbers of diamonds with higher carat amount.
